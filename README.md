## 소프트웨어의 기능

### 1. 로그인
> 로그인 설명
* 회원가입을 통해 ID 생성
* 사용자의 정보를 서버에 저장하고 불러오는 키로 사용 (memberId)
### 2 마이페이지
> 마이페이지 설명
* 키, 몸무게, 나이 등의 사용자 정보를 표시하고, 수정할 수 있게 하였음
* 사용자가 원하는 체형에 따른 목표를 설정할 수 있게 하였음
* 일정 기간 동안의 성취도 통계와 (성공, 실패 횟수), 체중 변화 그래프를 표시함.
